# GhostBounties vlayer Client

Browser-based client for generating ZK-TLS proofs for GitHub actions.

## Usage

1. Open `index.html` in a browser
2. The page will read URL parameters to determine the bounty:
   - `bounty_id`: The bounty ID
   - `action`: The GitHub action (star, fork, merge, comment, issue)
   - `repo_owner`: Repository owner
   - `repo_name`: Repository name
   - `reward`: Reward amount

3. Click "Generate Proof" to create a ZK-TLS proof
4. Copy the proof ID and submit it to the GhostBounties bot

## Integration

The client is accessed via a link generated by the XMTP agent:
```
https://prover.ghostbounties.io/?bounty_id=1&action=star&repo_owner=ghost-op&repo_name=core&reward=5%20USDC
```

## API

The `prover.js` file provides:
- `proveGitHubAction(action, repoOwner, repoName, prOrIssueNumber)`: Generate a proof
- `verifyProof(proofId)`: Verify a proof ID
