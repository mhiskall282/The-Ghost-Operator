# Images Directory

Place your images in this directory with the following names:

1. **hooded-figure.png** - The hooded figure with purple neon glow
   - Used in: Hero section (main visual)
   - Location: `images/hooded-figure.png`

2. **rabbit-drift.png** - The rabbit character with "DRIFT" text and neon green accents
   - Used in: Features section (Privacy First card)
   - Location: `images/rabbit-drift.png`

3. **rabbit-skull.png** - The rabbit character with sunglasses and skull t-shirt
   - Used in: Features section (Chat-Native card) and How It Works section
   - Location: `images/rabbit-skull.png`

4. **futuristic-character.png** - The futuristic humanoid character with cyan/blue glowing accents
   - Used in: Features section (Fully Automated card)
   - Location: `images/futuristic-character.png`

## Image Requirements

- Recommended format: PNG with transparent background
- Recommended size: 
  - Hero image: 500-800px width
  - Feature images: 200-300px width
  - Character images: 250-400px width
- All images will be automatically styled with purple glow effects and animations
